# Memory Clean

Clean up project memory:

1. Use `check-memory` to load current state
2. Use `memory-status` to see file sizes and counts
3. Identify outdated, duplicated, or contradictory information
4. Remove dead topic files and update MEMORY.md accordingly
5. Keep the system lean — if a topic file hasn't been relevant for 30 days, consider removing it
